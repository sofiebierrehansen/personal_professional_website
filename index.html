For the second semester exam, my group and I teamed up with Woofydays.com. The goal was to deliver Woofydays a website design that was more coherent with their DNA and vision for the company. Santa's sleigh - Game animations

During the first semester, we were given the challenge to create our own game design in order to learn Javascript and graphic design skills. I went with a christmas themed game with kawaii flat design. All graphics are made using shapes in Adobe Illustrator.

Oplev is a made-up travel guide that my group and I created as part of a dynamic webdesign project. We've created the logo in Adobe Illustrator and all the pictures are carefully selected to match our design which is in a classic complimentary colorpalette. See the website here.
